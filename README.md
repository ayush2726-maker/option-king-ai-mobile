# Option King AI Mobile

Mobile control app for Option King AI phone server.

## APK Build

This repo builds a standalone release APK with GitHub Actions.

1. Open the Actions tab.
2. Run `Build Android APK` or wait for the latest pull request build.
3. Download artifact `OptionKingAI-standalone-release-apk`.

Default server URL: `http://100.118.218.98:8765`
Default token: `optionking-local`

Standalone release build trigger: 2026-05-18
